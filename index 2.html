1. ماهية لغة HTML وأهميتها

تعتبر لغة HTML (HyperText Markup Language) هي لغة التوصيف العالمية التي تشكل العمود الفقري للشبكة العنكبوتية. هي ليست لغة برمجة إجرائية، بل هي لغة هيكلية تُعلم المتصفحات بكيفية عرض النصوص، الصور، الوسائط المتعددة، والجداول بصورة منطقية.

بدون HTML، لن يكون هناك إنترنت كما نعرفه اليوم. فهي اللغة التي تربط المليارات من المستندات عبر الروابط التشعبية (Hyperlinks).

2. تاريخ التطور والإصدارات

عام 1991: البداية
أطلق السير تيم بيرنرز لي أول نسخة تحتوي على 18 وسماً فقط.
عام 1995: HTML 2.0
أول معيار قياسي تم وضعه من قبل لجنة IETF.
عام 1999: HTML 4.01
النسخة التي صمدت لأكثر من عقد من الزمان ورسخت مفاهيم الويب.
عام 2014: ثورة HTML5
الإصدار الحالي الذي ندعمه اليوم، والذي أضاف ميزات الفيديو والصوت والرسم.

3. الدليل الشامل لوسوم HTML (A-Z)

سنستعرض هنا جميع الوسوم مقسمة حسب وظائفها التقنية:

وسوم النصوص الأساسية

  • <h1> إلى <h6>: عناوين بأهمية متدرجة.
  • <p>: لتعريف الفقرات النصية الطويلة.
  • <span>: حاوية صغيرة لتنسيق كلمات معينة.
  • <strong>: لتمثيل النصوص الهامة (بخط غليظ).
  • <em>: لإمالة النص لغرض التأكيد.
  • <mark>: لتمييز النص بخلفية ملونة (فسفوري).

وسوم الميديا والوسائط

  • <img>: لإدراج الصور (يحتاج لخاصية src).
  • <video>: لعرض مقاطع الفيديو المباشرة.
  • <audio>: لتشغيل الملفات الصوتية.
  • <canvas>: لرسم الرسوميات ثنائية الأبعاد عبر البرمجة.
  • <iframe>: لعرض صفحة ويب داخل صفحة أخرى.

وسوم القوائم والجداول

  • <ul> و <ol>: قوائم غير مرتبة ومرتبة.
  • <li>: عنصر القائمة الفردي.
  • <table>: لإنشاء الجداول المعقدة.
  • <thead> و <tbody>: تنظيم أجزاء الجدول.

جدول السمات العالمية (Attributes)

السمةالوصفالمثال
idمعرف فريد للعنصرid="header-1"
classتصنيف لعناصر متعددةclass="btn-primary"
styleإضافة تنسيق CSS مباشرstyle="color:red"
titleنص يظهر عند مرور الفأرةtitle="شرح الوسم"
langتحديد لغة المحتوىlang="ar"

4. مثال تطبيقي للهيكل الكامل

<!DOCTYPE html>
<html lang="ar" dir="rtl">
<head>
    <meta charset="UTF-8">
    <meta name="description" content="بحث مدرسي">
    <title>صفحتي الأولى</title>
</head>
<body>
    <header>
        <h1>أهلاً بكم في موقعنا التعليمي</h1>
    </header>
    <main>
        <article>
            <h2>موضوع البحث</h2>
            <p>هذا النص كتبه جميل وبودوارة بشغف كبير.</p>
        </article>
    </main>
    <footer>
        <p>حقوق النشر محفوظة 2026</p>
    </footer>
</body>
</html>
